# Anishiki

> ENG

A Chrome extension for Anilibria users that adds a quick link to the corresponding anime page on Shikimori.

Stop wasting time on manual searches. The extension reads the anime slug from the Anilibria URL, finds the match in the Shikimori database, and inserts a **Шикимори** chip next to the release status on the page.

### Features

- **One-click navigation** — open the anime page on Shikimori directly from Anilibria.
- **Automatic matching** — uses the Shikimori API to find the right title.
- **Native UI** — a compact chip that blends into the Anilibria interface.

### Installation

1. Clone or download this repository.
2. Open `chrome://extensions` in Chrome or Edge.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the project folder.
5. Open any release page on [Anilibria](https://anilibria.top) — the chip should appear next to the status.

### Requirements

- Google Chrome 88+ or Microsoft Edge 88+ (Manifest V3).

> RUS

Расширение для пользователей Anilibria, которое добавляет быструю ссылку на соответствующий сериал в базе Shikimori.

Расширение читает slug из URL страницы Anilibria, находит совпадение через API Shikimori и вставляет чип **Шикимори** рядом со статусом релиза.

### Особенности

- **Быстрый переход** — открывайте страницу на Shikimori прямо с Anilibria.
- **Автоматический поиск** — поиск через API Shikimori.
- **Встроенный UI** — компактный чип в интерфейсе Anilibria.

### Установка

1. Склонируйте или скачайте репозиторий.
2. Откройте `chrome://extensions` в Chrome или Edge.
3. Включите **Режим разработчика**.
4. Нажмите **Загрузить распакованное расширение** и выберите папку проекта.
5. Откройте любую страницу релиза на [Anilibria](https://anilibria.top) — чип появится рядом со статусом.

### Требования

- Google Chrome 88+ или Microsoft Edge 88+ (Manifest V3).
